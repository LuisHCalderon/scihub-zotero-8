# Zotero Sci-Hub Plugin (XPI)

Plugin .xpi para Zotero 8 que permite descargar PDFs automáticamente desde Sci-Hub.

## Instalación

1. Descargar el archivo `zotero-scihub-zotero8.xpi` desde la carpeta `/build`
2. Abrir Zotero
3. Ir a Tools > Plugins
4. Arrastrar el archivo `zotero-scihub-zotero8.xpi` en esa ventana.

## Compatibilidad
- Zotero 8+
